# China-Martyrs-Network-Crawler

# 中华英烈网爬虫

一个用于爬取[中华英烈网](https://www.chinamartyrs.gov.cn)烈士信息的Python爬虫工具，支持断点续爬、自动去重、分片保存和中文表头输出。

## ✨ 特性

- 🔄 **断点续爬**：意外中断后可从上次位置继续
- 🎯 **智能去重**：基于烈士唯一标识符(GUID)自动去重
- 📦 **分片保存**：定期保存CSV文件，避免数据丢失
- 🇨🇳 **中文表头**：输出CSV使用中文字段名，更易理解
- ⏱️ **灵活限速**：可配置请求间隔和冷却时间
- 🛡️ **高容错性**：支持自动重试和指数退避策略
- 📊 **进度显示**：实时显示爬取进度条

## 📋 系统要求

- Python 3.7+
- 必需的Python包：
  - requests
  - pandas
  - tqdm
